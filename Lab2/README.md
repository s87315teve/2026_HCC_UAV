# 人本計算實驗(Lab2)

## 環境建置
- 進入Lab1建立好的虛擬環境
```bash
conda activate UAV_env
```
- 安裝會使用到的套件
```bash
pip install opencv-python ultralytics
```

- 執行`yolo_example.py`
```bash
python yolo_example.py
```

執行成功的話會看到畫面上有框框把辨識到的物件標示出來
